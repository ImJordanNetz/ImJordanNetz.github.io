Now make this thang sliiide

To do this, we want the slider to keep track of where out mouse is, and then update the width of the left page accordingly. Luckily JS makes this pretty easy for us.


Let's grab the left side element using getElementById and store it in a variable. Since our slider is controlled by our mouse, we'll create a function to handle the onmousemove event. Inside this function, all we need to do is calculate the width of the left side as a percentage based on the current position of the mouse. Once we have this value, we can simply assign it to the width attribute on our element.

        
        
            const left = document.getElementById("left-side");

            const handleMove = e => {
              left.style.width = `${e.clientX / window.innerWidth * 100}%`;
            }
        
            document.onmousemove = e => handleMove(e);
        
            document.ontouchmove = e => handleMove(e.touches[0]);
